## ✨ Why this server is different

This server is built to the **API Factory Premium MCP Spec** — see [`PREMIUM_SPEC.md`](./PREMIUM_SPEC.md). Every server in our catalog ships with:

- 🛡 **Consistent error envelope** — `{ code, message, hint, retryable }` so the LLM can self-correct without re-prompting.
- 🔁 **Dual-output responses** — human-readable text + structured `_meta` for clean tool chaining.
- 🔍 **Built-in introspection** — every server exposes `about`, `health`, and `examples` tools out of the box.
- 📚 **Rich tool descriptions** — every tool includes "When to use", "Args", and "Returns" sections that Claude/Cursor display verbatim.
- 📄 **Pagination contract** — list tools always page with `cursor` + `limit`.
- ⏳ **Progress notifications** — long ops report progress to clients that support it.
- 🧪 **Dry-run mode** — destructive tools accept `dry_run: true` for safe previews.
- 🛟 **`safe()`-wrapped handlers** — no uncaught throw can poison the JSON-RPC channel.

Audited and gated by an automated 10-check linter before publish.

## Install

### Quickstart (no clone)
```bash
npx -y github:kfb-rich/json-toolbox-mcp
```

### Claude Desktop (`claude_desktop_config.json`)
```json
{
  "mcpServers": {
    "json-toolbox-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "github:kfb-rich/json-toolbox-mcp"
      ]
    }
  }
}
```

### Cursor (`~/.cursor/mcp.json`)
```json
{
  "mcpServers": {
    "json-toolbox-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "github:kfb-rich/json-toolbox-mcp"

  }
}
```

### From source
```bash
git clone https://github.com/kfb-rich/json-toolbox-mcp.git
cd json-toolbox-mcp
npm install
node server.js
```
