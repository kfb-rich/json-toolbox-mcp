# JSON Toolbox

> Pure-JS MCP server that gives AI assistants tools to validat

Pure-JS MCP server that gives AI assistants tools to validate, query (JSONPath), diff, format, and minify JSON. No external APIs needed.

## Tools
- **ping** — Health check.
- **echo** — Echoes input message.

## Install

### Quickstart (no clone)
```bash
npx -y github:kfb-rich/json-toolbox-mcp
```

### Claude Desktop (`claude_desktop_config.json`)
```json
{
  "mcpServers": {
    "json-toolbox-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "github:kfb-rich/json-toolbox-mcp"
      ]
    }
  }
}
```

### Cursor (`~/.cursor/mcp.json`)
```json
{
  "mcpServers": {
    "json-toolbox-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "github:kfb-rich/json-toolbox-mcp"
      ]
    }
  }
}
```

### From source
```bash
git clone https://github.com/kfb-rich/json-toolbox-mcp.git
cd json-toolbox-mcp
npm install
node server.js
```
